**Что реализовано по функциональности:**
1. Блог на чистом PHP 8.1+, Smarty, MySQL
2. Категории и статьи с полями, пагинация, сортировка
3. Главная страница с последними постами по категориям
4. Страницы категорий и отдельных постов с хлебными крошками
5. Пагинация и сортировка статей на странице категории
6. Похожие посты на странице статьи
7. Заглушки для изображений, если их нет

**По части кода**

Контроллеры:
1. MainController — главная
2. CategoryController — категория
3. PostController — пост

Фронт-контроллер:
1. Все маршруты проходят через public/index.php
2. 404 для любых неизвестных URL

Request-класс:
1. Обработка GET параметров с валидацией и дефолтами

Singleton для базы (Database):
1. Подключение через PDO
2. Параметры берутся из .env

Сидинг:
1. Скрипт для заполнения базы тестовыми категориями и статьями (10 категорий × 10 постов) - php src/Seedr.php
